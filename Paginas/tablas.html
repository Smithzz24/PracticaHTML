Tablas en HTML

Las tablas en HTML permiten organizar información en filas y columnas. Son útiles para mostrar datos estructurados como listas de estudiantes, productos, horarios o resultados. Las etiquetas principales utilizadas para crear tablas son <table>, <tr>, <th> y <td>.


Código:

<table border="1">

    <tr>
        <th>Nombre</th>
        <th>Edad</th>
        <th>Ciudad</th>
    </tr>
    <tr>
        <td>Ana</td>
        <td>22</td>
        <td>Bogotá</td>
    </tr>
    <tr>
        <td>Luis</td>
        <td>25</td>
        <td>Medellín</td>
    </tr>

</table>

<table>

La etiqueta <table> se utiliza para crear tablas en HTML. Una tabla está compuesta por filas y columnas.

<tr> → define una fila
<th> → define una celda de encabezado
<td> → define una celda de datos

Las tablas permiten organizar información de forma clara y estructurada dentro de una página web.
