Texto y Contenido

Las etiquetas de texto en HTML permiten dar estructura, significado y estilo al contenido escrito dentro de una página web.

<h1> - <h6>

Se utilizan para definir títulos y subtítulos según su nivel de importancia.

Código:

<h1>Ejemplo H1</h1>
<h2>Ejemplo H2</h2>
<h3>Ejemplo H3</h3>
<h4>Ejemplo H4</h4>
<h5>Ejemplo H5</h5>
<h6>Ejemplo H6</h6>

Ejemplo:

Ejemplo H1

Ejemplo H2

Ejemplo H3

Ejemplo H4

Ejemplo H5
Ejemplo H6

<p>

Define un bloque de texto o párrafo.

Código:

<p>Este es un párrafo de ejemplo en HTML.</p>

Ejemplo:

Este es un ejemplo de un párrafo en HTML.

<strong>, <b>, <em>, <i>

Permiten aplicar negrita o cursiva con o sin importancia semántica.

Código:

<strong>Texto importante</strong>

<b>Texto en negrita</b>

<em>Texto enfatizado con em</em>

<i>Texto en cursiva con i</i>

Ejemplo:

Texto importante con strong

Texto en negrita con b

Texto enfatizado con em

Texto en cursiva con i

<mark>, <small>, <del>, <ins>

Sirven para resaltar, reducir tamaño o indicar cambios en el texto.

Código:

<mark>Texto resaltado</mark>

<small>Texto pequeño</small>

<del>Precio anterior: $50</del>

<ins>Precio actual: $30</ins>

Ejemplo:

Texto resaltado

Texto pequeño

Texto eliminado

Texto insertado

<sub> y <sup>

Permiten escribir texto como subíndice o superíndice.

Código:


                                H<sub>2</sub>O
 10<sup>2</sup>

Ejemplo:

Fórmula: H2O

Potencia: 102

<blockquote> y <q>

Se utilizan para citas en bloque o en línea.

Código:

<blockquote>
  La educación es el arma más poderosa.
</blockquote>

<q>El conocimiento es poder</q>

Ejemplo:

Esta es una cita en bloque.

Ejemplo en línea: El conocimiento es poder

<code> y <pre>

Permiten mostrar fragmentos de código respetando formato y espacios.

Código:

<code>console.log("Hola");</code>

<pre>
<html>
  <body>Hola Mundo</body>
</html>
</pre>

Ejemplo:

Código en línea: console.log("Hola");

<html>
  <body>
    Hola Mundo
  </body>
</html>
