Estructurales

Normalmente una página web o aplicación web está dividida en diferentes áreas visuales para poder mejorar la experiencia del usuario y su interactividad. Las palabras clave que representan cada elemento nuevo de HTML5 están relacionadas a estas áreas visuales.

<header>

Representa la cabecera de una página o sección.

<nav>

Define una sección de navegación con enlaces.

<main>

Contiene el contenido principal del documento.

<section>

Representa una sección temática dentro de la página.

<article>

Contenido independiente como noticias o publicaciones.

<aside>

Contenido secundario como barras laterales.

<footer>

Pie de página con información adicional.

Ejemplos en codigo

Define la cabecera de una página o sección.

<header>
  <h1>Mi Página Web</h1>
  <p>Bienvenido a mi sitio</p>
</header>

Contiene enlaces de navegación.

<nav>
  <ul>
    <li><a href="#">Inicio</a></li>
    <li><a href="#">Servicios</a></li>
  </ul>
</nav>

Contiene el contenido principal del documento.

<main>
  <h2>Contenido Principal</h2>
  <p>Aquí va la información más importante.</p>
</main>

Define una sección temática dentro de la página.

<section>
  <h3>Noticias</h3>
  <p>Últimas actualizaciones del sitio.</p>
</section>

Representa contenido independiente.

<article>
  <h3>Artículo de Blog</h3>
  <p>Este es un artículo independiente.</p>
</article>

Contiene información complementaria.

<aside>
  <h4>Publicidad</h4>
  <p>Contenido adicional o enlaces relacionados.</p>
</aside>

Define el pie de página.

<footer>
  <p>© 2026 Mi Sitio Web</p>
  <a href="#">Términos y Condiciones</a>
</footer>
