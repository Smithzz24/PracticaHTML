Formularios

Los formularios en HTML permiten al usuario enviar datos a un servidor o interactuar con la página. Son la base de registros, búsquedas, encuestas y mucho más; aquí verás ejemplos claros y bien diseñados con Bootstrap.

¿Cómo funciona cada tipo?

A continuación se muestran los principales tipos de entrada que puedes utilizar en un formulario HTML. Todos los controles están decorados con clases de Bootstrap y envueltos en bloques genéricos para mantener el estilo.

Nunca compartiremos su correo electrónico con nadie más.







Usa <input type="text"> para texto corto.

<input type="text" class="form-control" placeholder="Ingrese texto">

El tipo email valida automáticamente el formato.

<input type="email" class="form-control" placeholder="user@example.com">

Usa type="password" para ocultar el texto.

<input type="password" class="form-control" placeholder="Contraseña">

Usa <input type="checkbox"> o radio para opciones. 

<input type="checkbox" id="check1"> <label for="check1">Aceptar</label>
<input type="radio" name="gender" value="M"> Masculino

Etiqueta <select> con opciones <option>.

<select class="form-select">
  <option>Opción 1</option>
  <option>Opción 2</option>
</select>

Usa <textarea> para textos largos o comentarios.

<textarea class="form-control" rows="3"></textarea>

El botón type="submit" envía el formulario; otros tipos son button y reset. 

<button type="submit" class="btn btn-primary">Enviar</button>
