Formularios

Los formularios son la forma estándar en HTML para recopilar información de los usuarios. A continuación se muestran todos los tipos de controles disponibles; cada ejemplo incluye el código y el resultado en la página.

Controles de formulario

Navega por las secciones de la derecha para saltar rápidamente a un tipo de control.

Texto

<input type="text" class="form-control" placeholder="Texto corto">

Correo electrónico

<input type="email" class="form-control" placeholder="user@example.com">


Contraseña

<input type="password" class="form-control" placeholder="Contraseña">

Número

<input type="number" class="form-control" min="0" max="100">

Teléfono

<input type="tel" class="form-control" placeholder="+56 9 1234 5678">

URL

<input type="url" class="form-control" placeholder="www.ejemplo.com">

Búsqueda

<input type="search" class="form-control" placeholder="Buscar...">

Color

<input type="color" class="form-control form-control-color" value="#563d7c">

Fecha y hora

<input type="date" class="form-control"> (Sirve para saber la fecha comun)
                                

<input type="time" class="form-control"> (Sirve para saber la hora comun)
                                

<input type="datetime-local" class="form-control"> (Sirve para saber la fecha y hora local)
                                

<input type="month" class="form-control"> (Sirve para saber el mes y año)
                                

<input type="week" class="form-control">
(Sirve para saber la semana y año)




Checkbox y radio

<input type="checkbox" id="check1"> <label for="check1">Opción 1</label>

<input type="radio" name="group" id="rad1" value="a"> <label for="rad1">A</label>


Archivo

<input type="file" class="form-control">

Oculto

Los campos hidden no se muestran en la página.

<input type="hidden" name="token" value="123ABC">

No visible en el navegador.

Menú desplegable

<select class="form-select">
  <option>Rojo</option>
  <option>Verde</option>
  <option>Azul</option>
</select>

Área de texto

<textarea class="form-control" rows="3"></textarea>

Datalist

<input list="browsers" class="form-control">
<datalist id="browsers">
  <option value="Chrome">
  <option value="Firefox">
  <option value="Safari">
</datalist>

Rango

<input type="range" class="form-range" min="0" max="10">

Botones

<button type="submit" class="btn btn-primary">Enviar</button>
<button type="reset" class="btn btn-secondary">Reset</button>
<button type="button" class="btn btn-success">Clic</button>
