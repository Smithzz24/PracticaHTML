Multimedia

Las etiquetas multimedia en HTML permiten integrar contenido audiovisual, gráfico e interactivo dentro de una página web.

<img>

Permite insertar imágenes en una página web.

Atributos comunes:
  • src
  • alt
  • width
  • height
  • loading

<audio>

Permite insertar archivos de audio.

Atributos comunes:
  • controls
  • autoplay
  • loop
  • muted
  • preload

<video>

Permite insertar videos.

Atributos comunes:
  • controls
  • autoplay
  • loop
  • muted
  • poster
  • width / height

<source> y <track>

Se utilizan dentro de audio o video para definir formatos alternativos y subtítulos.

  • source: src, type
  • track: src, kind, srclang, label

<figure> y <figcaption>

Agrupan contenido multimedia con una descripción.

<canvas>

Permite dibujar gráficos dinámicos usando JavaScript.

<svg>

Permite crear gráficos vectoriales.

<iframe>

Permite insertar contenido externo como videos o mapas.

  • src
  • width
  • height
  • allowfullscreen

<code> y <pre>

Permiten mostrar fragmentos de código respetando formato y espacios.

Código en línea: <img src="imagen.jpg" alt="Descripción">

<video controls>
  <source src="video.mp4" type="video/mp4">
</video>

Ejemplos en codigo

<img src="foto.jpg" alt="Descripción de la imagen"
width="300" height="200">

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Tu navegador no soporta audio.
</audio>

<video controls width="400" poster="preview.jpg">
  <source src="video.mp4" type="video/mp4">
  Tu navegador no soporta video.
</video>

<video controls>
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  <track src="subtitulos.vtt" kind="subtitles"
         srclang="es" label="Español">
</video>

<figure>
  <img src="imagen.jpg" alt="Paisaje">
  <figcaption>Descripción de la imagen</figcaption>
</figure>

<canvas id="miCanvas" width="300" height="150">
</canvas>

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40"
          stroke="black" fill="lightblue"/>
</svg>

<iframe width="560" height="315"
src="https://www.youtube.com/embed/ID"
allowfullscreen>
</iframe>
