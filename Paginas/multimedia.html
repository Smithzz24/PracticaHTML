Multimedia

Las etiquetas multimedia en HTML permiten integrar contenido audiovisual, gráfico e interactivo dentro de una página web.

<img>

Permite insertar imágenes en una página web.

Atributos comunes:
  • src
  • alt
  • width
  • height
  • loading

Código:

<img src="foto.jpg" alt="Descripción de la imagen" width="300" height="200">

Ejemplo:

<audio>

Permite insertar archivos de audio.

Atributos comunes:
  • controls
  • autoplay
  • loop
  • muted
  • preload

Código:

<audio controls> <source src="audio.mp3" type="audio/mpeg"> Tu navegador no soporta audio. </audio>

Ejemplo:

<video>

Permite insertar videos.

Atributos comunes:
  • controls
  • autoplay
  • loop
  • muted
  • poster
  • width / height

Código:

<video controls width="400" poster="preview.jpg"> <source src="video.mp4" type="video/mp4"> Tu navegador no soporta video. </video>

Ejemplo:

<source> y <track>

Se utilizan dentro de audio o video para definir formatos alternativos y subtítulos.

  • source: src, type
  • track: src, kind, srclang, label

Código:

<video controls> <source src="video.mp4" type="video/mp4"> <source src="video.webm" type="video/webm"> <track src="subtitulos.vtt" kind="subtitles" srclang="es" label="Español"> </video>

Ejemplo:

Estos elementos se usan dentro de <audio> o <video> para proporcionar múltiples formatos y subtítulos.

<figure> y <figcaption>

Agrupan contenido multimedia con una descripción.

Código:

<figure> <img src="imagen.jpg" alt="Paisaje"> <figcaption>Descripción de la imagen</figcaption> </figure>

Ejemplo:

<canvas>

Permite dibujar gráficos dinámicos usando JavaScript.

Código:

<canvas id="miCanvas" width="300" height="150"> </canvas>

Ejemplo:

<svg>

Permite crear gráficos vectoriales.

Código:

<svg width="100" height="100"> <circle cx="50" cy="50" r="40" stroke="black" fill="lightblue"/> </svg>

Ejemplo:

<iframe>

Permite insertar contenido externo como videos o mapas.

  • src
  • width
  • height
  • allowfullscreen

Código:

<iframe width="560" height="315" src="https://www.youtube.com/embed/ID" allowfullscreen> </iframe>

Ejemplo:

<code> y <pre>

Permiten mostrar fragmentos de código respetando formato y espacios.

Código:

<p>Código en línea: <code>&lt;img src="imagen.jpg" alt="Descripción"&gt;</code></p> <pre><code>&lt;video controls&gt; &lt;source src="video.mp4" type="video/mp4"&gt; &lt;/video&gt;</code></pre>

Ejemplo:
