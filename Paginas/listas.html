Listas en HTML

Las listas en HTML permiten organizar información de manera estructurada. Son fundamentales para menús, pasos, características de productos, glosarios y estructuras jerárquicas dentro de una página web.

¿Qué tipos de listas existen?

HTML permite crear tres tipos principales de listas: desordenadas, ordenadas y de definición. También es posible anidarlas entre sí.

Lista Desordenada (ul)

<ul> <li>HTML</li> <li>CSS</li> <li>JavaScript</li> </ul>
  • HTML
  • CSS
  • JavaScript

Lista Ordenada (ol)

<ol> <li>Paso 1</li> <li>Paso 2</li> <li>Paso 3</li> </ol>
  1. Paso 1
  2. Paso 2
  3. Paso 3

Lista de Definición (dl)

<dl> <dt>HTML</dt> <dd>Lenguaje de marcado</dd> </dl>
HTML
Lenguaje de marcado
