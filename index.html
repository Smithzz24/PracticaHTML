Etiquetas HTML

Las etiquetas HTML son los elementos básicos que se utilizan para estructurar y dar formato a una página web. Cada etiqueta tiene un propósito específico y se utiliza para definir diferentes partes del contenido.


Para crear una etiqueta se abre con un menor que < y se pone la etiqueta deseada. Luego se cierra con un mayor que >. Para cerrar la etiqueta se usa </>.

Ejemplo de etiquetas HTML:

    <!DOCTYPE html>

<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, 
        initial-scale=1.0">
    <title>Ejemplo de Etiquetas HTML</title>
</head>

<body>
    <h1>Hola Mundo</h1>
    <p>Este es un párrafo de ejemplo.</p>
    <a href="https://www.ejemplo.com">Enlace a Ejemplo.com</a>
</body>

</html>

Resultado:

Hola Mundo

Este es un párrafo de ejemplo.

Enlace a Ejemplo.com

Tipos de etiquetas

Existen diferentes tipos de etiquetas HTML, cada una con su función específica. Algunas de las más comunes incluyen

Estructura

Son las etiquetas semánticas que organizan y dividen el contenido de una página web, como <header>, <main>, <section>, <article> y <footer>.

Listas

Permiten organizar información en formato ordenado o desordenado mediante etiquetas como <ul>, <ol> y <li>.

Tablas

Se utilizan para mostrar datos organizados en filas y columnas usando etiquetas como <table>, <tr>, <td> y <th>.

Texto

Incluyen etiquetas para mostrar contenido textual como títulos, párrafos y formatos de énfasis, por ejemplo <h1> a <h6>, <p>, <strong> y <em>.

Formularios

Permiten recopilar información del usuario mediante elementos como <form>, <input>, <textarea>, <select> y <button>.

Multimedia

Permiten integrar contenido audiovisual e interactivo como imágenes, audio y video utilizando etiquetas como <img>, <audio>, <video> e <iframe>.
