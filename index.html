Etiquetas HTML

Las etiquetas HTML son los elementos básicos que se utilizan para estructurar y dar formato a una página web. Cada etiqueta tiene un propósito específico y se utiliza para definir diferentes partes del contenido.


Para crear una etiqueta de abre con un Menor que (<) y se pone la etiequeta deseada. Posteriormente se ponen todos los atributos e identificadores necesarios. Luego se cierra con un Mayor que (>). Con esto la etiqueta queda creada, luego se pone el contenido dentro de este, texto, imagenes o todo lo que sea necesario. Para cerrar la etiqueta luego del contenido se abre con un slash (/) seguido de la etiqueta de apertura. Es decir, se usa la misma etiqueta puesta antes.

Ejemplo de etiquetas HTML:

    <!DOCTYPE html>

<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ejemplo de Etiquetas HTML</title>
</head>

<body>
    <h1>Hola Mundo</h1>
    <p>Este es un párrafo de ejemplo.</p>
    <a href="https://www.ejemplo.com">Enlace a Ejemplo.com</a>
</body>

</html>
